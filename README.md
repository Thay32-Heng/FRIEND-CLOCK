# ⏳ Friend-Clock — The Ultimate Squad Celebration Hub

**Friend-Clock** is a high-energy, dark-themed birthday tracker designed to make sure you never miss a friend's big day. It’s more than just a countdown; it’s a full party experience with automated dashboards, music, and social coordination.

## 🚀 Top Features

* **Smart Birthday Detection**: The "Brain" automatically scans your squad list and detects who is celebrating today.
* **Hero Dashboard**: On a friend's birthday, the site transforms! Their photo and name take center stage at the top of the screen.
* **Infinite Party Loop**: Features a "Start Birthday Music" button that plays the "Happy Birthday" song on a continuous loop to set the vibe.
* **Hotpot Protocol (KH Style)**: A specialized invitation button that generates a ready-made Telegram message to invite the whole squad for a hotpot dinner.
* **Zodiac Integration**: Every friend’s card displays their Zodiac sign (Aries, Pisces, etc.) based on their birth date.
* **Friendship Stats**: A sleek footer that tracks "Squad Size," identifies who is "Partying Now," and predicts the "Next Party."

## 🛠️ Tech Stack

* **Frontend**: HTML5, CSS3 (Modern Flex/Grid), and Vanilla JavaScript.
* **Animations**: Canvas-Confetti API for birthday blasts.
* **Hosting**: GitHub Pages.
